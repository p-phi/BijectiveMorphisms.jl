# Bijectivism — Flip it, chain it, own it [![Build Status](https://github.com/p-phi/Bijectivism.jl/actions/workflows/CI.yml/badge.svg?branch=main)](https://github.com/p-phi/Bijectivism.jl/actions/workflows/CI.yml?query=branch%3Amain)

> Type-safe, no math
> Lean back, no fat
> Dispatch, on track
> Biject, no slack

**Bijectivism.jl** lets you treat every transformation as a first-class reversible citizen. Compose bijections like building blocks, invert them on demand, and ensure your pipelines always roundtrip cleanly. Lean, type-safe, and Julian-friendly — because forward, backward, and composed should feel effortless.
